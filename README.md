## things i've built

| project | what it does |
|:---|:---|
| [m0viesentiment](https://github.com/Maaadhavq/m0viesentiment) | sentiment NLP on movie reviews — the one that made me realize I actually liked this stuff |
| [Ai-job-finder](https://github.com/Maaadhavq/Ai-job-finder) | matches job listings to your skills using ML — built it because job searching is painful |
